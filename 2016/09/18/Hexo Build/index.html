Hexo Build

2016-09-18
Hexo

I will post what I learned in my blog. This article indicated how to build Hexo in Github.

Intro

There are lots of system to create a website. I choose Hexo because it was created by Taiwanese. Rather than Jekyll which created by Ruby, Hexo was developed by Node.js and it seems faster than Jekyll.

Requirement

You should install Git and Node.js at first.

Node.js

Step

  1. install Hexo

  2. create directory

  3. install hexo package

  4. Set Github account

  5. Create SSH key to deploy

  6. Set SSH key on Github Github -> account setting -> SSH Public Keys -> add another public keys

  7. Open _config.yml
